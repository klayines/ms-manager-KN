# ms-manager-KN
Manager of your minecraft servers, from Klayines Network. Create your own servers with full control and convenient management, the manager is available in web format and as an application for windows .exe format. Each server has its own directorate and is independent of each other. The project is completely free. 
To use the web version of the manager, unzip the downloaded archive into a separate folder and run the file "app.py " and follow the link to the local website that appears for managing and creating servers.
